    第1章 Roughness と Resolution の違い ▾
    プロローグ

    「ガサガサ」と「ピッチがぼやける」は別もの

    高い声を出した時、耳の中で 2つの違う現象 が同時に起きている。
    この章のゴールは、その2つを区別する2軸 ― Roughness と Resolution ― を 見て・体感して 理解すること。

    §1

    声は「階段」でできている

    あなたが 220 Hz の声を出すと、耳には 220, 440, 660, 880, … Hz の音が同時に届いている。これが 倍音 (H1, H2, H3, …)。

    倍音の周波数 Hn = n × f₀
    §2

    この階段、上にいくほど耳の中でギュッと縮む

    Hz では 等間隔。でも耳の感じる距離（cent）では上ほど狭くなる。

    H1→H21200cOct
    H2→H3702cP5
    H3→H4498cP4
    H4→H5386cM3
    H5→H6316cm3
    H6→H7267c<m3
    H7→H8231c<m3
    H8→H9204cM2
    H9→H10182c<M2
    H10→H11165c<M2

    Note cent 値は f₀ に依らない。バスでもソプラノでも H5→H6 は 316c。

    仕掛け①

    耳の中の「臨界帯域」— Roughness の正体

    耳には ERB (臨界帯域) という「音を分ける小窓」が並ぶ。
    同じ窓に2つの倍音が入ると、耳は ダブった1音 として感じる ― これが ザラつき (Roughness) の正体 (Plomp & Levelt, 1965)。

    ERB の幅 (Glasberg & Moore, 1990) ERB(f) = 24.7 × (4.37·f/1000 + 1) Hz
    Rough になる条件 f₀ ≤ ERB(n·f₀)
    仕掛け②

    ピッチに使われる倍音には限界がある — Resolution の正体

    耳は すべての倍音をピッチの手がかりにはしていない。
    ある番号より上は「聞こえてはいるけど音程の判別には使われない」状態になる (Plomp, 1967)。
    これが Resolution の話。

    Resolved (ピッチに統合される) 条件 n < 9 かつ n·f₀ < 5000 Hz
    §5

    ここが山場 — 2軸は別の話

    RoughnessResolution
    なに？音色の ザラつき音程として 分離できるか
    原因隣接倍音が ERB 内に入るn ≥ 9、または ≥ 5 kHz
    f₀ への依存依存する（連続）依存しない（離散）
    境界の動きf₀ で 動く常に 同じ番号
    聞こえ方buzzy / 濃い音程としては浮かない

    重要 男声の H6〜H8 はかなり Rough だが、ピッチには統合される（Rough & Resolved）。「ザラついている」と「音程がぼやけている」がイコールではない実例。

    §6

    2軸を AND すると、4ゾーンを通り過ぎていく

    体感

    スライダーで f₀ を動かして、4ゾーンを目で追う

    — Hz ←いまの声の高さ (f₀)

    目安: 話し声 ≈ 100〜250 Hz / 歌の高音 ≈ 500 Hz 以上。

    あなたの倍音マップ (H1〜H16)
    各行 = 1 倍音（H番号 / Hz / ゾーン）。 行間の ) 部分 = 2倍音間のインターバル（cent + 度数、f₀ 非依存）。

    ザラザラゾーンRough & Unresolved

    いまの声では—

    5 kHz より上。音程の判断には使われず、「輝き / sheen」として音色に貢献するだけ。

    まとめ

    3行でおさらい

    1. Roughness = 隣りあう倍音が ERB に入るか。f₀ で動く・連続。
    2. Resolution = その倍音がピッチに使われるか。n=9 と 5 kHz の固定境界。
    3. AND で 4 ゾーン: Pure / R&R / R&U / Ceiling。

    Key ザラついている（Rough）と、音程として聞かれない（Unresolved）は別の話。

    第2章 フォルマントと音色 — Source-Filter モデル ▾
    プロローグ

    同じ「ア」でも、人によって違う音色。なぜ？

    声の高さ (f₀) は 声帯 が決める。
    でも「ア / イ / ウ」の違いや「明るい / 暗い」音色は、声帯じゃなくて 声道（口の中の形） が決めている。
    ここに Source-Filter モデル という考え方がある。

    §1

    声は「音源 × フィルタ」の掛け算

    Key 音源（声帯）が 素材 を提供し、フィルタ（声道）が 選んで強調 する。

    §2

    フォルマント = 声道の共鳴ピーク

    声道（喉〜口腔〜唇）の形が変わると、特定の周波数が共鳴して山ができる。
    この山を フォルマント (F1, F2, F3, F4, F5) と呼ぶ。

    • F1 ≈ 300〜850 Hz: 母音の「開き具合」(/i/=狭い → /a/=広い)
    • F2 ≈ 800〜2500 Hz: 母音の「前後」(/u/=後ろ → /i/=前)
    • F3 ≈ 2.0〜3.0 kHz: 音色の明るさ・個人性
    • F4, F5: Singer's Formant の構成要素（後で出てくる）
    §3

    F1-F2 平面 — 母音は地図で示せる

    母音F1 (Hz)F2 (Hz)感じ
    /i/ ee≈ 300≈ 2300狭く前
    /e/ eh≈ 500≈ 1900半広・前
    /a/ ah≈ 750≈ 1200広く中央
    /o/ oh≈ 500≈ 900半広・後
    /u/ oo≈ 350≈ 800狭く後
    §4

    Open / Close Timbre — 2f₀ と F1 の関係

    f₀ を上げていくと、ある瞬間 2f₀ が F1 を超える。
    この境界を Turnover といい、音色が「開いた (Open)」から「閉じた (Close)」へ切り替わる (Bozeman)。

    §5

    Singer's Formant — オペラの「飛ぶ声」の秘密

    プロのオペラ歌手は、F3 + F4 + F5 が 2.5〜3.5 kHz に集中する独特の山を持つ。
    この山 (Singer's Formant) はオーケストラの隙間（耳が最も敏感な帯域）に乗るため、伴奏を超えて遠くまで届く (Sundberg, 1974)。

    Note Singer's Formant は 低い喉頭位置 と 狭い咽頭 によって F3/F4/F5 が物理的に近接して生まれる現象。

    まとめ

    3行でおさらい

    1. 声 = 音源（声帯）× フィルタ（声道） の掛け算。
    2. F1, F2 で母音が決まる。F3〜F5 は音色・個人性・遠達性。
    3. 2f₀ が F1 を超えると Turnover、Open → Close Timbre へ移行する。
    参考文献
    • Fant, G. (1960). Acoustic Theory of Speech Production. Mouton.
    • Sundberg, J. (1974). Articulatory interpretation of the singing formant. JASA, 55(4), 838–844.
    • Bozeman, K. W. Practical Vocal Acoustics. NATS Books.
    • Titze, I. R. Principles of Voice Production. NCVS.
    第3章 発声モード — Flow / Pressed / Breathy ▾
    プロローグ

    同じ音程・同じ母音でも、声には「性格」がある

    スピーチでも歌でも、同じ「ラ (220Hz) のア」を出しているのに、
    ある人は 豊かでバランスの取れた 声、別の人は 突き刺さる buzzy な声、また別の人は 息漏れの多い柔らかい 声に聞こえる。
    この違いは 声門の閉じ方（phonation）で決まる。

    §1

    3つの力のバランス: P, R, U

    §2

    3つの発声モード — 抵抗 R の取り方の違い

    モード声門波形音色
    Flow完全閉鎖だが過剰でない滑らか・対称豊か・自然・伸びる
    Pressed接触率↑・閉鎖鋭い急峻な閉鎖・狭い openbuzzy・突き刺さる
    Breathy不完全閉鎖・空気漏れ滑らか・底が浮く柔らかい・息っぽい
    §3

    Rd パラメータで一本につながる

    Fant (1995) の Rd 形状パラメータ は、Pressed → Modal → Breathy を 1つの数 で表現する。
    Rd は LF モデルの「波形の形」を決める一括スイッチ。

    < 0.5
    Pressed
    0.5–0.9
    やや Pressed
    0.9–1.3
    Modal
    1.3–2.0
    やや Breathy
    > 2.0
    Breathy
    §4

    スペクトル傾斜で見分ける

    波形の違いは、スペクトルの 傾き (slope) として現れる。
    高次倍音が どれだけ落ちないか でモードが判別できる。

    §5

    音楽的にはどう使う？

    • Flow: クラシック・ベルカントの基本。最も伸びる。
    • Pressed: ロック・ベルティング・叫び声。耳に強く届くが過剰だと声帯への負担↑。
    • Breathy: 親密・繊細な歌い回し。ジャズや R&B のニュアンス表現。長時間は不経済。

    Caution Pressed 寄りは 声帯への機械的負荷 が高い。長時間の使用や練習なしの多用は声帯結節などの原因に。

    まとめ

    3行でおさらい

    1. 発声は P / R / U（圧・抵抗・気流）のバランス。U = P/R。
    2. R の取り方 で Flow / Pressed / Breathy の3モードが決まる。
    3. Rd パラメータが3モードを 1つの数 で表現し、スペクトル傾斜にも対応する。

    • Fant, G. (1995). The LF-model revisited. STL-QPSR, 36(2-3), 119–156.
    • Titze, I. R. (1994). Principles of Voice Production. Prentice Hall.
    • Bozeman, K. W. Practical Vocal Acoustics. NATS Books.
