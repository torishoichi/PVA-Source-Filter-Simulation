声楽音響 — 式をベースに

なぜ管の長さで Z が変わるのか
kL → tan → 共鳴 f = c/4L

手応え Z は Zin = j·Z0·tan(kL)。このうち管の長さ L はどこに効くのか。kL を入口に、tan の正体、そして共鳴の式 f = c/4L までを式ベースでたどります。

問い — 長さはどこに効くのか

式のどこを見ればいい

Zin = j · Z0 · tan(kL)

この式に「長さ L」は1か所しか出てきません ── tan(kL) の中の kL。だから「長さで Z が変わる」を理解する道は1本：L → kL → tan(kL) → Z。

左の j は「90度ずらす印」、Z0 は手応えの土台 ── どちらにも L は入っていません。L が握っているのは tan(kL) ただひとつ。順に kL とは何か → tan が何をするか → 長さを変えると何が動くか と進みます。

1 — kL ＝「管の中に波がどれだけ入っているか」

入口の記号

k = ω/c。ω は声の角周波数（どれだけ速く振動するか）、c は音速。k は「波が1メートルあたり何ラジアン進むか」＝波の細かさです。それに管の長さ L を掛けた kL は ──

kL の意味

kL ＝ いまその管の中に「波がどれだけ入っているか」。同じ高さの声（ω が同じ＝k が同じ）でも、管が長ければ kL は大きく、短ければ小さい。L が直接 kL を動かす ── これが長さの効く入口。

図1 — 同じ声でも、長い管ほど kL は大きい

波の細かさ（k）は同じ＝同じ高さの声。それでも管が長いほど、中に入る波の量＝kL は大きくなる。これが tan(kL) を動かす。

2 — tan(kL) ＝ kL しだいで表情を変える関数

Z の値を決める部分

kL が決まると、tan(kL) が Z の値そのものを決めます。tan は kL の大きさで表情を3段階に変えます。

  • kL が小さい（＝波の細かさ k と長さ L の積が小さいとき）→ tan(kL) ≈ kL、小さなプラス → Z は小さくイナーティブ。
  • kL が π/2 に近づく → tan(kL) → ∞、爆発 → ここが共鳴（フォルマント）。管にちょうど 1/4 波長が入った瞬間。
  • kL が π/2 を超える → tan はマイナスに反転 → Z はコンプライアント。
注意 — kL は「声の高さ」ではない

声の高さを決めるのは k だけ（k が大きい＝高い声）。kL は k × L ── 長さ L が掛かっているので「高さそのもの」ではなく「管に入る波の量」です。だから「kL が小さい＝低い声」とは言えません（短い管なら、高い声でも kL は小さい）。tan と Z を決めるのは、声の高さではなく kL。

図2 — tan(kL) の3つの表情

この曲線が Z の正体。kL が右へ進むほど、tan は『小さなプラス → 爆発 → マイナス』と表情を変える。詳細編で見た「下＝イナーティブ／上＝コンプライアント」は、この tan 一文字に書き込まれている。

ここで気づくはずです。「長さ L を変える」とは、kL を変えること。そして kL を変えるとは、この tan の曲線の上を、左右に移動すること。次はそれを動かしてみます。

3 — 長さを変える＝tan の上を動く

まず図の読み方

ここからは「周波数の地図」で見ます。下の図は3つの部品でできています ── 先に読み方を説明します。

  • 上＝声道の管。長さ L。この管の第一共鳴は F1 = c/4L。L を変えると、この F1 が動きます。
  • 下＝手応え Z の曲線。横軸は周波数、縦軸は手応え（上＝助ける／イナーティブ、下＝じゃま／コンプライアント）。図2の tan を周波数の軸に並べたものです。F1 のすぐ下は「助ける」、すぐ上は「じゃま」。さらに管は F1 の奇数倍（F2≒3F1、F3≒5F1…）でも共鳴するので、同じ模様がくり返します。
  • 黒い縦線＝声の倍音。ここでは倍音を1本だけ 700 Hz に固定します ── 声は変えず、管の長さだけ動かす実験です。この倍音が曲線のどの帯に乗っているか（黒丸の位置）が「いまこの声は助けられているか／じゃまされているか」。

曲線が F1 などで切れて上から下へ飛ぶのは、tan が爆発（∞）して反対側（−∞）から戻るからです。切れ目＝共鳴の位置。

スライダーで管を長くしてみてください。F1 = c/4L なので F1 が下がり、曲線全体が左（低い周波数）へずれます。声の倍音（黒い縦線）は動かないのに、曲線のほうがずれて、黒丸が「助ける」帯と「じゃま」帯のあいだを移っていく ── これが「同じ声でも、管の長さで手応え Z が変わる」ということです。

4 — 共鳴の式 f = c / 4L を導く

「爆発」を周波数で書く

曲線がずれる量を、式で正確に出せます。tan が爆発する条件は kL = π/2。これを周波数 f について解くだけです。

kL = π/2　→　(2πf/c)·L = π/2　→　f = c / 4L

途中で使ったのは k = ω/c = 2πf/c だけ。出てきた f = c/4L が共鳴（第一フォルマント）の周波数です。読み取れることは2つ。

  • 管が長い（L 大）ほど、共鳴の周波数 f は低い。分母に L があるから。
  • 共鳴は c/4L の奇数倍にも並ぶ（3c/4L, 5c/4L…＝F2, F3）。L を変えれば、それらがいっせいに動く。

数値で：音速 c ≈ 343 m/s、声道の長さ L ≈ 17.5 cm なら f = 343 / (4×0.175) ≈ 490 Hz。これが「Z の曲線が L を変えると左右にずれる」の正体 ── ずれ幅は f = c/4L がきっちり決めている。

5 — 長さ L と 太さ A の役割分担

式の中の住み分け

式 Zin = j·Z0·tan(kL) をもう一度見ると、管の形は2か所に入っています ── 長さ L は tan(kL) に、太さ A は Z0 = ρc/A に。役割はきれいに分かれます。

長さ L → tan(kL) を通じて、共鳴がどの周波数に来るかを決める（f = c/4L）。曲線の左右の位置。

太さ A → Z0 を通じて、手応えの大きさの土台を決める（太いほど Z₀ 小＝弱い）。曲線の縦の高さ。A は共鳴の位置を動かさない。

つまり「長さで Z が変わる」と「太さで Z が変わる」は、別々の効き方。長さは曲線を横にずらし、太さは曲線を縦に伸び縮みさせる。今回の主題は前者 ── 横のずれです。

6 — だから SOVT は共鳴を下げる

式どおりのこと

最後に、これまでの章とつながる話。ストロー発声やリップトリル（SOVT）は、唇の先に管を継ぎ足すようなもの ── 実効的に L が長くなります。

あとは f = c/4L の通り。L が増える → f が下がる → Z の曲線が丸ごと左へずれる。SOVT で第一共鳴 F1 が下がるのは、感覚的な話ではなく、この式がそう言っているからです。ストローを長くするほど効果が強いのも、L が効くから。

「管の長さで Z が変わる」を一文で ── L が変わる → kL が変わる → tan(kL) の上を動く → 共鳴が c/4L に従って移り、Z の曲線ごと左右にずれる。

まとめ

一本道だった

「管の長さで Z が変わる」の道は、式 Zin = j·Z0·tan(kL) の中で1本だけ ──

L 変わる → kL 変わる → tan(kL) の上を動く → 共鳴 f = c/4L が移り、Z 曲線ごとずれる

j も Z0 も長さを含まない。長さが触れるのは tan(kL) ただひとつで、その効きめは f = c/4L がきっちり決める。太さ A は別担当（手応えの大きさ）。SOVT が共鳴を下げるのも、実効的に L を伸ばしているから ── ぜんぶ同じ一本道の話でした。

用語

Zin管が声に返す手応え（入力インピーダンス）。Zin=j·Z0·tan(kL)。
j「90度ずらす」印。リアクタンス（損なく貯めて返す手応え）のしるし。長さは含まない。
Z0 = ρc/A手応えの土台の大きさ。太さ A で決まる。長さ L は含まない。
k（波数）k=ω/c。波が1メートルあたり何ラジアン進むか＝波の細かさ。
ω（角周波数）声がどれだけ速く振動するか。声の高さで決まる。
c（音速）空気中を音が進む速さ。およそ 343 m/s。
L（管長）声道の長さ。kL を通じて共鳴の位置を決める。
A（断面積）管の太さ。Z0 を通じて手応えの大きさを決める。共鳴の位置は動かさない。
kL「管の中に波がどれだけ入っているか」。L が大きいほど kL も大きい。
tan(kL)kL しだいで Z の値を決める部分。小さい時プラス、π/2 で爆発、その上でマイナス。
共鳴／フォルマントtan が爆発する高さ（kL=π/2）。管が特に響かせる周波数。
f = c/4L第一共鳴の周波数。管が長いほど低い。Z 曲線のずれ幅を決める式。
イナーティブ／コンプライアント共鳴より下＝イナーティブ（助ける）、上＝コンプライアント（じゃま）。tan の符号で決まる。
SOVTストロー等で出口を狭める練習。実効的に L を伸ばし、f=c/4L どおり共鳴を下げる。
