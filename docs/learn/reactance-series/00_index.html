声楽音響 — 学習パック

リアクタンス解説シリーズ
目次と、読む順番

「なぜ倍音と共鳴の関係で声道の手応え（リアクタンス）が変わるのか」を、やさしい順に積み上げた7本。番号の順に読むと、たとえ話から非線形ソース・フィルター結合まで一段ずつ上がれます。

本編ラダー（01 → 06 の順に）

下にいくほど深くなります。むずかしく感じたら、その手前で止めても筋は通ります。

01導入編 — はじめてのリアクタンスやさしい

数式なし、たとえ話だけ。管が声を「助ける／じゃまする」とはどういうことかを、ブランコや重い空気のイメージでつかむ。
02中級編 — 手応えを「曲線」で見る中級

用語を導入。倍音列・フォルマント（F1/F2）・CQ/PTP。手応えを周波数の「くり返す曲線」として捉え、パッサッジョのターンを位置づける。
03詳細編 — なぜ符号が反転するのか詳細

物理と数式で「なぜ」を詰める。X = ωM − 1/ωC と開管の tan(kL)、周波数領域と時間領域（ブランコ）の対応、CQ↑・PTP↓まで。
04上級編 — 非線形ソース・フィルター結合上級

厳密なインピーダンス、仕事ループ W=∮p dA、流パルスの歪み、Level 1/2 結合、分岐、声門下系と epilarynx。研究レベルの全体像。
05上級編・やさしい版 — 数式を友だちにする上級／やさしく

04 と同じ内容を、削らずに再解説。Zin=j·Z0·tan(kL) も W=∮p dA も、記号を1つずつ注釈図で友だちにする。
06上級編やさしい版 vol2 — 2つの式をつなぐ上級／やさしく

Zin の式と W の式の「つなぎ」を細かく。橋の式 p = Zin·U、j と位相 φ の同一性、波とループが連動するインタラクティブ。

補足（本編のどこからでも）

07補足 — なぜ管の長さで Z が変わるのか補足

L → kL → tan(kL) → 共鳴 f = c/4L を式ベースで。03〜06 を読む途中で「長さの効き方」が気になったら、ここへ。
おすすめの歩き方 ── はじめてなら 01 → 02 で全体像。数式に踏み込みたくなったら 03。さらに 04 が重ければ、05 → 06 で同じ内容をゆっくり。07 はいつでも参照できる補足です。各ファイルは単体で完結し、互いの閲覧順に依存しません。
