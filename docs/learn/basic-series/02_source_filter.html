フォルマントと音色 — Source-Filter モデル

プロローグ

同じ「ア」でも、人によって違う音色。なぜ？

声の高さ (f₀) は 声帯 が決める。でも「ア / イ / ウ」の違いや「明るい / 暗い」音色は、声帯じゃなくて 声道（口の中の形） が決めている。 ここに Source-Filter モデル という考え方がある。

§1

声は「音源 × フィルタ」の掛け算

Key 音源（声帯）が 素材 を提供し、フィルタ（声道）が 選んで強調 する。

§2

フォルマント = 声道の共鳴ピーク

  • F1 ≈ 300〜850 Hz: 母音の「開き具合」(/i/=狭い → /a/=広い)
  • F2 ≈ 800〜2500 Hz: 母音の「前後」(/u/=後ろ → /i/=前)
  • F3 ≈ 2.0〜3.0 kHz: 音色の明るさ・個人性
  • F4, F5: Singer's Formant の構成要素
§3

F1-F2 平面 — 母音は地図で示せる

母音F1 (Hz)F2 (Hz)感じ
/i/ ee≈ 300≈ 2300狭く前
/e/ eh≈ 500≈ 1900半広・前
/a/ ah≈ 750≈ 1200広く中央
/o/ oh≈ 500≈ 900半広・後
/u/ oo≈ 350≈ 800狭く後
§4

Open / Close Timbre — 2f₀ と F1 の関係

f₀ を上げていくと、ある瞬間 2f₀ が F1 を超える。 この境界を Turnover といい、音色が「開いた (Open)」から「閉じた (Close)」へ切り替わる (Bozeman)。

§5

Singer's Formant — オペラの「飛ぶ声」の秘密

プロのオペラ歌手は、F3 + F4 + F5 が 2.5〜3.5 kHz に集中する独特の山を持つ。この山はオーケストラの隙間（耳が最も敏感な帯域）に乗るため、伴奏を超えて遠くまで届く (Sundberg, 1974)。

まとめ

3行でおさらい

  1. 声 = 音源（声帯）× フィルタ（声道） の掛け算。
  2. F1, F2 で母音が決まる。F3〜F5 は音色・個人性・遠達性。
  3. 2f₀ が F1 を超えると Turnover、Open → Close Timbre へ移行する。
