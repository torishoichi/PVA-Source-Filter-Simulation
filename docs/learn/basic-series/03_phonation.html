基本シリーズ — 第3章

発声モード — Flow / Pressed / Breathy

プロローグ

同じ音程・同じ母音でも、声には「性格」がある

同じ「ラ (220Hz) のア」を出しているのに、ある人は 豊かでバランスの取れた 声、別の人は 突き刺さる buzzy な声、また別の人は 息漏れの多い柔らかい 声に聞こえる。この違いは 声門の閉じ方（phonation）で決まる。

§1

3つの力のバランス: P, R, U

§2

3つの発声モード — 抵抗 R の取り方の違い

モード声門波形音色
Flow完全閉鎖だが過剰でない滑らか・対称豊か・自然・伸びる
Pressed接触率↑・閉鎖鋭い急峻な閉鎖・狭い openbuzzy・突き刺さる
Breathy不完全閉鎖・空気漏れ滑らか・底が浮く柔らかい・息っぽい
§3

Rd パラメータで一本につながる

Fant (1995) の Rd 形状パラメータ は、Pressed → Modal → Breathy を 1つの数 で表現する。

< 0.5
Pressed
0.5–0.9
やや Pressed
0.9–1.3
Modal
1.3–2.0
やや Breathy
> 2.0
Breathy
§4

スペクトル傾斜で見分ける

§5

音楽的にはどう使う？

  • Flow: クラシック・ベルカントの基本。最も伸びる。
  • Pressed: ロック・ベルティング・叫び声。耳に強く届くが過剰だと声帯への負担↑。
  • Breathy: 親密・繊細な歌い回し。長時間は不経済。

Caution Pressed 寄りは 声帯への機械的負荷 が高い。長時間の使用や練習なしの多用は声帯結節などの原因に。

まとめ

3行でおさらい

  1. 発声は P / R / U（圧・抵抗・気流）のバランス。U = P/R。
  2. R の取り方 で Flow / Pressed / Breathy の3モードが決まる。
  3. Rd パラメータが3モードを 1つの数 で表現し、スペクトル傾斜にも対応する。
