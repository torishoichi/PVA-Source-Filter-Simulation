🎬 0. WAVEFORMって何を見る画面？
ピヨ猫

WAVEFORM のタブ、波がウニョウニョ動いたり、ピタッと止まったり、たまに高速でパカパカする。あれって全部“正常”なの？

ピヨ鳥

いい入り口だピヨ！WAVEFORM は時間領域のオシロスコープピヨ。

スペクトラムが「周波数の絵」なら、こっちは耳に実際に届く圧力波形を時間で見る場所ピヨ。

今日おさえる背骨はこの3つだピヨ：

  • 静止して見えるのは、トリガーが波形に位相同期しているから
  • 純正律（有理数比）は静止、平均律（無理数比）はゆっくり変化が正しい
  • 高速なパカパカ切り替わり“だけ”がバグ

この3つが今日の背骨ピヨ！

ピヨ猫

なるほど…“止まってる＝正常”とは限らないってことニャ？

ピヨ鳥

その通りピヨ。“いつ止まって、いつ変わるのが正しいか”を、順番にほどいていこうピヨ！

🗺 1. 問いの地図

🌳 中心の問い：WAVEFORM が「止まる／変わり続ける」のは、いつ正しくて、いつバグ？
— 答え：トリガーが波形に位相同期していれば正常。比が有理数なら静止、無理数なら“ゆっくり変化”が物理的に正しく、高速パカパカだけがバグ —

  • 🖥 問1　WAVEFORM は何を映し、何本の線が出る？

    答え：時間領域のオシロで、青=メイン・teal=ハモリ・violet=合成波の3本。

    • 時間領域オシロ 用語耳に届く圧力波形を時間軸で見る画面。SPECTRUM と対をなす。
    • 合成波＝各時点の線形和 事実violet は青＋teal の和で、実際に鳴っている音そのもの。
    • violet を描く条件 主張synth＋ハモリON のときだけ。Mic・ハモリOFF では描かない。

  • 🎯 問2　動く波がなぜ止まって見える？

    答え：毎フレーム“立ち上がりゼロ交差”に表示窓を吸着させるから。

    • 立ち上がりゼロ交差トリガー 用語実機オシロと同じ、同じ位相で窓を切る仕組み。
    • 相関トリガー 事実前フレームに最も似た候補を選び、合成波の位相も固定する。
    • ちらつきバグ 主張素朴に“最初の交差”を拾うと候補間を往復して発生する。

  • ⚖️ 問3　いつ静止でき、いつ変わり続けるのが正しい？

    答え：純正律（有理数比）は真に周期的で静止、平均律（無理数比）は非周期で変化が正しい。

    • 共通周期＝周期の最小公倍数 用語220Hz と 110Hz なら合成波の真の周期は 9.1ms。
    • Oct は平均律でも 2:1 事実厳密な整数比なので、平均律でも Oct だけは静止する。
    • 高速パカパカ＝常にバグ 主張物理は“ゆっくり変化”まで。高速な往復は実装の誤り。

  • 🌊 問4　画面の“形の変化”は何を教える？

    答え：近接部分音のうなりが形の変化として見え、干渉で振幅が正直に振れる。

    • うなり（beat） 用語近い周波数差でできる、周期的な強弱の脈。
    • 8.7Hz のうなり 事実平均律長3度で 1100Hz と 1108.7Hz が約 8.7Hz でうなる。
    • 合成波ピーク基準の共有スケール 主張建設的干渉の最大約2倍まで、3本を正直に比較。
ピヨ猫

上の“問いの地図”って、どう読むの？

ピヨ鳥

資料の論理を“問いのリレー”で1枚にした地図ピヨ。前の枝の答えが、次の枝の問いを生むピヨ。

読み方は、帯（見出し）＝問い、すぐ下の一行＝答え。色タグは 用語=青・事実=紫・主張=赤ピヨ。

問1→問4の“答え”だけ拾い読みすれば、WAVEFORM の理屈が上から下へ流れるピヨ！

① 2. 時間領域 vs 周波数領域
図1：同じ音の“2つの顔”

左＝時間の揺れ／右＝倍音の内訳。どちらも「同じ1つの音」を別の軸で見ているだけ。WAVEFORM は耳に届く圧力そのものを映す。

ピヨ猫

SPECTRUM の倍音の棒グラフはもう慣れてる。WAVEFORM はそれと何が違うの？

ピヨ鳥

同じ音を“別の軸”で見てるだけピヨ。SPECTRUM は横軸が周波数、WAVEFORM は横軸が時間ピヨ。

仕様でも、その役割はこう言ってるピヨ：

「WAVEFORM は時間領域のオシロスコープ。スペクトラム（周波数領域）と対をなし、『耳に実際に届く圧力波形』を見る場所。」仕様①
ピヨ猫

じゃあ WAVEFORM は“耳のマイクが拾う生の揺れ”そのものってことね。

ピヨ鳥

そうピヨ。SPECTRUM が“成分表”なら、WAVEFORM は“実際の圧力の時間変化”ピヨ。だから「今どんな音か」はここに全部出るピヨ！

② 3. 3本の線＝重ね合わせ
ピヨ猫

線が3色出るときがある。青・緑っぽいの・紫。あれは何を表してるの？

ピヨ鳥

色で役割が違うピヨ：

  • 青＝メイン声（Source→Tilt→F1..F5 を通過した後）
  • teal＝ハモリ声部（同じフィルタチェーンを通過した後）
  • violet＝合成波（青＋teal の各時点の和）

violet の定義は、仕様ではこうピヨ：

「violet=合成波（青+teal の各時点の和 = 実際に鳴っている音）」仕様②
ピヨ猫

なるほど。じゃあ紫は“実際に鳴ってる音”そのものなんだ。

ピヨ鳥

その通りピヨ。だから紫が主役、青と teal はその部品ピヨ。

ハモリ OFF なら合成≡メインだから、紫は描かない（青1本が正常）ピヨ。Mic のときも紫は出さないピヨ。

Mic で紫を消す理由は、仕様でこう言ってるピヨ：

「Mic 時も violet 非表示（青 trace が mic になり成分関係が成立しないため）。」仕様②
③ 4. オシロのトリガー
図2：毎フレーム、同じ位相に窓を合わせる → 波が静止して見える

緑の点＝波形の“立ち上がりゼロ交差”。毎フレーム、ここに表示窓の左端を吸着させると、波は動いていても切り取る位相が揃い、静止画になる。

ピヨ猫

動く波が止まって見えるって…もしかして、こっそり一時停止ボタンを押してるだけ？

ピヨ鳥

ピヨピヨピヨヨヨヨ、止め絵じゃないピヨ！

毎フレーム、波形の立ち上がりゼロ交差に表示窓を合わせてるピヨ。実機オシロスコープと同じ原理ピヨ。

仕様の言葉ではこうピヨ：

「毎フレーム、波形の『立ち上がりゼロ交差』に表示窓を吸着させる（実機オシロスコープと同じ原理）。周期波形なら毎回同じ位相で切れる → 静止して見える。」仕様③
ピヨ猫

同じ位相で切るから、周期波なら毎回同じ形になって、止まって見えるのね。

ピヨ鳥

まさにそれピヨ。波は動いてるのに、切り取る位相を揃えるから“静止画”になるピヨ。図の緑の点が、毎フレーム合わせにいく立ち上がりゼロ交差ピヨ！

④ 5. 共通周期とちらつき（実際に起きたバグ）
図3：メイン220Hz＋Oct↓110Hz — 真の周期は9.1ms、候補は4.5ms毎

メインのゼロ交差は4.5ms毎（縦線＝トリガー候補）。だが合成波の真の周期は9.1ms。緑（0/9/18ms）では合成波が一致、赤（4.5/13.5ms）ではハモリの位相がちょうど反転している。

図4：素朴なトリガーが行き来する“2つの形”＝ちらつきの正体

同じ音でも、切り出す位相が違えば別の形になる。素朴に“最初の交差”を拾うと窓Aと窓Bをフレーム毎に往復＝ちらつき。相関トリガーは前フレームに似た方（＝いつも窓A）を選び続けて固定する。

ピヨ猫

トリガーがあるのに、なんでパカパカ暴れることがあったの？

ピヨ鳥

ハモリを足すと、話が一段むずかしくなるピヨ。

メイン220Hz（周期4.5ms）に Oct↓110Hz（周期9.1ms）を重ねると、合成波の“真の周期”は9.1msピヨ。両方の周期の最小公倍数（共通周期）ピヨ。

でもトリガー候補＝メインのゼロ交差は4.5ms毎にあるピヨ。図3の縦線ピヨ。

ピヨ猫

4.5msごとに候補があるのに、真の周期は9.1ms…半分ズレた候補があるってこと？

ピヨ鳥

鋭いピヨ！1個おきの候補（赤）では、ハモリの位相がちょうど反転してるピヨ。

素朴に“最初の交差を拾う”と、フレーム毎に緑と赤のどちらを拾うかがランダムピヨ。だから2つの形を高速で行き来＝ちらつきピヨ。

図4の窓Aと窓Bが、その2つの形ピヨ。

ピヨ猫

同じ音なのに、切る位置で別の形…これがパカパカの正体か。じゃあ直し方は？

ピヨ鳥

あるべき解決は相関トリガーピヨ：

「相関トリガー: 前フレームの合成波と最も似た候補を選び、メインのロックを保ったまま合成波の位相も固定する。」仕様④
ピヨ猫

メインのロックは保ったまま、合成波の位相も固定するのね。

ピヨ鳥

そうピヨ。“青は青で止めつつ、紫も同じ形で止める”ピヨ。これでちらつきは消えるピヨ！

⑤ 6. 純正律 vs 平均律（教材のキモ）
図5：純正律 Oct（2:1）— 合成波は9.1msごとに“完全に同じ形”＝静止

純正律 Oct は比が厳密な 2:1（有理数）。合成波は 9.1ms ごとに完全に一致（＝）するので、トリガーで切れば静止する。

ピヨ猫

純正律だと止まって、平均律だと止まらないって聞いた。なんで“律”で変わるの？

ピヨ鳥

比が“有理数か無理数か”で決まるピヨ。

純正律は周波数比が 3/2・5/4・6/5・2/1 みたいな有理数ピヨ。だから合成波は真に周期的で、静止できるピヨ。図5がOct（2:1）で、9.1ms ごとに完全に同じ形ピヨ。

ピヨ猫

じゃあ平均律で止まらないのは…画面がバグって壊れてるってこと？

ピヨ鳥

ちがうピヨ、そこが今日いちばん大事ピヨ！

平均律は完全5度でも比が 2(7/12)≈1.4983 の無理数ピヨ。合成波は厳密には二度と同じ形にならない＝非周期ピヨ。

だから、仕様ではこう言い切ってるピヨ：

「平均律（Oct以外）= 比が無理数（例 完全5度 2^(7/12)≈1.4983）→『完全静止』は原理的に不可能で、ゆっくり形が変わり続けるのが物理的に正しい姿。」仕様⑤
ピヨ猫

“完全静止”は無理で、ゆっくり変わり続けるのが正しい姿…。オクターブだけ別なの？

ピヨ鳥

Oct だけは特別ピヨ。平均律でも Oct は厳密に整数比だから、静止するピヨ：

「Oct だけは平均律でも厳密に 2:1 なので静止する。高速なパカパカ切り替わりは常にバグであり物理ではない。」仕様⑤
ピヨ猫

まとめると、ゆっくり変化＝物理、高速パカパカ＝バグ、だね。

ピヨ鳥

その線引きが命ピヨ。ゆっくり形が変わるのは近接倍音が干渉してる証拠で、正しい表示ピヨ。高速な行き来だけを疑えばいいピヨ！

⑥⑦ 7. うなり・干渉・振幅
図6：平均律の長3度 — 1100Hz と 1108.7Hz が 8.7Hz でうなる

1100Hz（=5×220）と 1108.7Hz（=4×277.18）が約8.7Hzでうなる。振幅の“節（打ち消し）”が周期的に来て、波形の形がゆっくり脈打つ。※搬送波(~1100Hz)は密集のため省略、包絡線のみ図示。

ピヨ猫

その“ゆっくりの変化”って、具体的に何が見えてるの？

ピヨ鳥

うなり（beat）が“形の変化”として見えてるピヨ。仕様の例で言うとこうピヨ：

「近接部分音のうなり（例: 平均律長3度 277.18Hz では 5×220=1100Hz と 4×277.18=1108.7Hz が約8.7Hzでうなる）が『形の変化』として見える。」仕様⑤
ピヨ猫

1100Hz と 1108.7Hz で 8.7Hz…上の図の“くびれ”がその節ね。

ピヨ鳥

そうピヨ。近い周波数どうしが強め合ったり打ち消し合ったりして、振幅がゆっくり脈打つピヨ。図6のくびれが打ち消し（節）ピヨ。

ピヨ猫

合成波は建設的干渉で2倍まで振れるって…音量も2倍でご近所に怒られる？

ピヨ鳥

ピヨピヨピヨヨヨヨ、それは“表示の振れ幅”の話ピヨ！

縦軸の作りは、仕様でこう決めてるピヨ：

「合成波は建設的干渉で成分の和（最大約2倍）まで振れる → 縦軸は合成波ピーク基準の共有スケール（3本の相対振幅が正直に比較できる）。」仕様⑥
ピヨ猫

合成波ピーク基準の共有スケールだから、3本の相対の大きさが正直に比べられるのね。

ピヨ鳥

その通りピヨ。建設的干渉で膨らむ所も、破壊的干渉で凹む所も、同じ物差しで正直に出るピヨ！

⑦ 8. モード別・あるべき表
ピヨ猫

結局、どのモードでどんな絵が出てれば正常なの？一覧が欲しい。

ピヨ鳥

頭に入れておくと一発で異常判定できるピヨ。あるべき一覧はこうピヨ：

  • synth のみ → 青1本
  • synth ＋ Harmony → 青＋teal＋violet
  • Mic → 橙赤（violet なし）
  • 録音再生 → 青・プレイヘッド中心・violet なし
  • 長窓ズームアウト → 包絡線＋うねり(beat)読み出し

この一覧と違う絵が出たら、そこが疑いどころピヨ！

❓ 9. FAQ
ピヨ猫

青が1本しか出てない。壊れてる？

ピヨ鳥

壊れてないピヨ。ハモリ OFF なら合成≡メインだから、紫は出さず青1本が正常ピヨ。Mic 時も紫は出ないピヨ！

ピヨ猫

マイクで歌ったら紫が消えた。なんで？

ピヨ鳥

Mic 時は青 trace がマイク波形そのものになって、青＝メイン合成の“成分関係”が成り立たないからピヨ。だから合成波（紫）は非表示が正しいピヨ！

ピヨ猫

ズームアウトしたら波が包絡線みたいになった。

ピヨ鳥

それも正常ピヨ。長い窓では1画素に何周期も入るから、min/max の包絡線＋うねり(beat)読み出しに切り替わるピヨ！

ピヨ猫

平均律でゆっくり形が変わる。直すべき？

ピヨ鳥

直さなくていいピヨ。無理数比だから非周期で、ゆっくり変化が物理的に正しい姿ピヨ。高速パカパカだけがバグピヨ！

✅ 10. たしかめクイズ＆まとめ
ピヨ猫

最後に理解チェックしたい。

ピヨ鳥

たしかめクイズだピヨ！

Q. 220Hz に、平均律の完全5度（比 2(7/12)≈1.4983）を重ねた。合成波は“静止”する？

静止しないピヨ。比が無理数で合成波は非周期だから、ゆっくり形が変わり続けるのが正しい姿ピヨ。もし高速でパカパカ切り替わったら、それはトリガーのバグピヨ。

まとめ：トリガーは“同位相で切る道具”。有理数比なら静止、無理数比ならゆっくり変化が正解、高速パカパカだけがバグ。3本の線（青＝メイン／teal＝ハモリ／violet＝合成波）を、合成波ピーク基準の共有スケールで正直に読むピヨ！
