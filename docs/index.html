詳細解説 — cent とは / buzzy 感との関係

cent とは / なぜ cent で並べるか

cent＝音と音の距離を「耳の感じ方」で測る単位。Hz は物理的な差、cent は 比（何倍か）。100 cent = 半音、1200 cent = 1 オクターブ。

倍音は Hz だと等間隔（隣との差はいつも f₀）。でも耳は等間隔とは感じない — H1→H2 は 2倍 = 1200c、H10→H11 は 1.1倍 = 165c。差はどちらも f₀ Hz なのに、上に行くほど耳の中でギュッと詰まる。

f₀ 非依存性と buzzy 感の本体

f₀ 非依存: cent 値は比 (n+1)/n だけで決まり、f₀ に依らない。バス声でもソプラノでも H5→H6 は同じ 316c。声の高低を問わず効く普遍スケール。

buzzy の本体: cent が縮むこと自体がラフネスを生むのではない。cent が縮むことで 隣接倍音が同じ ERB に入り込む のがラフの本体（Plomp & Levelt 1965）。

高周波では ERB ≈ 187 cents でほぼ一定になるため、「約 200c 以下の音程はラフ閾値」 という近似が使える。典型 f₀ では H8→H9 = 204c がちょうど境界、H9→H10 = 182c から確実にラフ帯。

詳細解説 — 分類の仕組みと ERB

分類は2軸の AND

  • Roughness 軸（f₀ 依存・連続的）— 隣接倍音が同じ ERB に入れば Rough。f₀ を変えると境界が動く。上端1段目の帯が対応。
  • Resolution 軸（離散・固定）— Hn ≥ 9 の pitch dominance cutoff、および Hn·f₀ > 5 kHz の天井。f₀ に依らず常に同じ境界。上端2段目の帯が対応。

この2軸を AND で重ねて Pure / Rough & Resolved / Rough & Unresolved / Ceiling の4ゾーンに分類。

ERB = Equivalent Rectangular Bandwidth

人の聴覚フィルタを矩形で近似したときの実効幅。同じ ERB 内に入る2音は、聴覚にとって1音として処理される（臨界帯域）。

Glasberg & Moore (1990) の近似式: ERB(f) = 24.7 × (4.37 f/1000 + 1) Hz

H 番号は離散表示だが、実際の遷移は連続的。上端 Roughness 帯のグラデーションがそれを表現している。

Based on: Practical Vocal Acoustics: Pedagogic Applications for Teachers and Singers — Kenneth Bozeman (NATS Books)

第1章 Roughness と Resolution の違い ▾
プロローグ

「ガサガサ」と「ピッチがぼやける」は別もの

高い声を出した時、耳の中で 2つの違う現象 が同時に起きている。 この章のゴールは、その2つを区別する2軸 ― Roughness と Resolution ― を 見て・体感して 理解すること。

§1

声は「階段」でできている

あなたが 220 Hz の声を出すと、耳には 220, 440, 660, 880, … Hz の音が同時に届いている。これが 倍音 (H1, H2, H3, …)。

倍音の周波数 Hn = n × f₀
§2

仕掛け①

耳の中の「臨界帯域」— Roughness の正体

耳には ERB (臨界帯域) という「音を分ける小窓」が並ぶ。 同じ窓に2つの倍音が入ると、耳は ダブった1音 として感じる ― これが ザラつき (Roughness) の正体 (Plomp & Levelt, 1965)。

ERB の幅 (Glasberg & Moore, 1990)ERB(f) = 24.7 × (4.37·f/1000 + 1) Hz
Rough になる条件f₀ ≤ ERB(n·f₀)
仕掛け②

ピッチに使われる倍音には限界がある — Resolution の正体

耳は すべての倍音をピッチの手がかりにはしていない。 ある番号より上は「聞こえてはいるけど音程の判別には使われない」状態になる (Plomp, 1967)。これが Resolution の話。

Resolved (ピッチに統合される) 条件n < 9 かつ n·f₀ < 5000 Hz

第2章 フォルマントと音色 — Source-Filter モデル ▾
プロローグ

同じ「ア」でも、人によって違う音色。なぜ？

声の高さ (f₀) は 声帯 が決める。でも「ア / イ / ウ」の違いや「明るい / 暗い」音色は、声帯じゃなくて 声道（口の中の形） が決めている。 ここに Source-Filter モデル という考え方がある。

§1

声は「音源 × フィルタ」の掛け算

Key 音源（声帯）が 素材 を提供し、フィルタ（声道）が 選んで強調 する。

§2

フォルマント = 声道の共鳴ピーク

  • F1 ≈ 300〜850 Hz: 母音の「開き具合」(/i/=狭い → /a/=広い)
  • F2 ≈ 800〜2500 Hz: 母音の「前後」(/u/=後ろ → /i/=前)
  • F3 ≈ 2.0〜3.0 kHz: 音色の明るさ・個人性
  • F4, F5: Singer's Formant の構成要素
§3

F1-F2 平面 — 母音は地図で示せる

母音F1 (Hz)F2 (Hz)感じ
/i/ ee≈ 300≈ 2300狭く前
/e/ eh≈ 500≈ 1900半広・前
/a/ ah≈ 750≈ 1200広く中央
/o/ oh≈ 500≈ 900半広・後
/u/ oo≈ 350≈ 800狭く後
§4

Open / Close Timbre — 2f₀ と F1 の関係

f₀ を上げていくと、ある瞬間 2f₀ が F1 を超える。 この境界を Turnover といい、音色が「開いた (Open)」から「閉じた (Close)」へ切り替わる (Bozeman)。

§5

Singer's Formant — オペラの「飛ぶ声」の秘密

プロのオペラ歌手は、F3 + F4 + F5 が 2.5〜3.5 kHz に集中する独特の山を持つ。この山はオーケストラの隙間（耳が最も敏感な帯域）に乗るため、伴奏を超えて遠くまで届く (Sundberg, 1974)。

まとめ

3行でおさらい

  1. 声 = 音源（声帯）× フィルタ（声道） の掛け算。
  2. F1, F2 で母音が決まる。F3〜F5 は音色・個人性・遠達性。
  3. 2f₀ が F1 を超えると Turnover、Open → Close Timbre へ移行する。

  • Bozeman, K. W. Practical Vocal Acoustics. NATS Books.

第3章 発声モード — Flow / Pressed / Breathy ▾
プロローグ

同じ音程・同じ母音でも、声には「性格」がある

同じ「ラ (220Hz) のア」を出しているのに、ある人は 豊かでバランスの取れた 声、別の人は 突き刺さる buzzy な声、また別の人は 息漏れの多い柔らかい 声に聞こえる。この違いは 声門の閉じ方（phonation）で決まる。

§1

3つの力のバランス: P, R, U

§2

3つの発声モード — 抵抗 R の取り方の違い

モード声門波形音色

Rd パラメータで一本につながる

§4

スペクトル傾斜で見分ける

§5

音楽的にはどう使う？

  • Flow: クラシック・ベルカントの基本。最も伸びる。
  • Pressed: ロック・ベルティング・叫び声。耳に強く届くが過剰だと声帯への負担↑。
  • Breathy: 親密・繊細な歌い回し。長時間は不経済。

Caution Pressed 寄りは 声帯への機械的負荷 が高い。長時間の使用や練習なしの多用は声帯結節などの原因に。

まとめ

3行でおさらい

  1. 発声は P / R / U（圧・抵抗・気流）のバランス。U = P/R。
  2. R の取り方 で Flow / Pressed / Breathy の3モードが決まる。
  3. Rd パラメータが3モードを 1つの数 で表現し、スペクトル傾斜にも対応する。
